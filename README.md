# Берёзка — Ресторан русской кухни

Сайт ресторана русской кухни «Берёзка» в Москве.

## Стек

- **HTML5** — семантическая вёрстка
- **CSS3** — CSS-переменные, Grid, анимации
- **Alpine.js 3** — реактивное меню, форма бронирования, навигация
- **Google Fonts** — Lora (serif), Tenor Sans (display), Inter (body)

## Особенности

- Alpine.js: интерактивное меню с фильтром по категориям (5 категорий, 15 блюд)
- Alpine.js: форма бронирования с счётчиком гостей (+/−)
- Анимации появления через IntersectionObserver
- Анимированный герой с параллакс-скроллом
- Цитата основательницы с фоновым фото
- Фото-галерея в CSS Grid (разные размеры ячеек)
- Адаптивная навигация с бургером

## Разделы

1. **Герой** — полноэкранный, слоган, CTA
2. **История** — фото, история, награды
3. **Меню** — 5 категорий с Alpine.js tabs, 15 блюд с фото и ценами
4. **Цитата** — pull-quote на тёмном фоне
5. **Галерея** — CSS Grid с разными пропорциями
6. **Бронирование** — форма с Alpine.js + контакты

## Запуск

Откройте `index.html` в браузере. Alpine.js подключён через CDN — нужен интернет.
